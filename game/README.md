# Game Files

Ce dossier doit contenir le fichier `main.dol` du jeu original.

**Ce fichier n'est PAS inclus dans le repo** (contenu du jeu).

## Comment l'obtenir

```bash
python3 tools/extract_ciso.py votre_rom.ciso TP_Extracted
cp TP_Extracted/sys/main.dol game/main.dol
```

Puis ouvrez `game/main.dol` dans Dolphin pour jouer avec le mod.
